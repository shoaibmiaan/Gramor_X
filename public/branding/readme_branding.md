## Usage

<link rel="apple-touch-icon" href="/branding/gramorx-ai-appicon-1024.png">
<link rel="manifest" href="/branding/manifest.json">

Notes

- Use the animated SVG where motion is acceptable.
- Use the static SVG for emails, PDFs, or raster contexts.
- Monochrome variants are for high-contrast contexts.
